# Trabalho de Sistemas de Computação

## Regras

-   Não fazer push direto na `main`.
-   Criar branch separada para cada tarefa.
-   Enviar alterações via Pull Request (PR).
-   Apenas Yasmin pode aprovar merges.

## Como Contribuir

### Setup (1ª vez)

```bash
git clone URL_DO_REPOSITORIO_AQUI.git
cd NOME_DO_REPOSITORIO

git checkout main # Vai para a branch main
git pull origin main # Atualiza a branch main
git checkout -b nome-da-branch # Criar branch nova
```

### Salvar alterações

```bash
git add .
git commit -m "mensagem" # Mensagem: Alterações realizadas
git push origin nome-da-branch
```

### Pull Request

1. Acesse o repositório no GitHub.

2. Clique em "Compare & pull request".

3. Escreva título e descrição.

4. Clique em "Create pull request".

Aguarde a revisão e o merge.
